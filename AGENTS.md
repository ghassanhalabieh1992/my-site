# AGENTS.md

## Project Context

This is a React + Vite portfolio website for Nexora AI. All content is managed from `src/data/portfolio.js`.

## Key Files

- `src/` — frontend application source
- `src/data/portfolio.js` — all website content (projects, skills, experience, contact)
- `src/components/portfolio/` — portfolio section components
- `vite.config.js` — Vite configuration
- `.env.local` — local environment values; never commit secrets

## Development

- Run `npm run dev` for local development
- Run `npm run build` to build for production
- Run `npm run lint` to check for lint errors
