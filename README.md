# Nexora AI — Portfolio Website

A modern portfolio website showcasing software engineering and AI projects.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion

## Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Project Content

All website content (projects, skills, experience, contact info) is managed from a single file:

```
src/data/portfolio.js
```

Edit that file to update any content without touching component code.
